# MOBO for calibrating coarse-grained polymer miscibility
This repository contains the codebase for the framework presented in the paper "Assessing the accuracy of Bayesian-optimized CGMD in predicting polymer miscibility". 

Please cite the paper if you use the data presented here. Please access the paper at: [https://chemrxiv.org/doi/full/10.26434/chemrxiv-2025-p14dx/v2](https://chemrxiv.org/doi/full/10.26434/chemrxiv-2025-p14dx/v2)
